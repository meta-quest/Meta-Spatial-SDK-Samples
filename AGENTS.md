# Meta Spatial SDK — Samples

This repository contains sample projects for the **Meta Spatial SDK**, which builds spatial/MR apps for **Meta Quest VR/MR headsets** (not standard Android phone/tablet apps).

Each sample subdirectory is a standalone Gradle project with its own `AGENTS.md` describing the SDK architecture.
